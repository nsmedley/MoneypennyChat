How to set up a live chat from an email

Pre Chat Form (If enabled)

Place the script inside of the landing page where people will be landing from your email; for best results it is recommended that this goes as close as possible to the to the closing </body> tag. The url that they enter must contain "livechat=true" for example if your landing page is xyz.com/emailCampaign you will need to amend the link to xyz.com/emailCampaign?livechat=true

This script will open your chat as if the user has clicked on your livechat button so they will see a pre-chat form if you have it enabled.

<script type="text/javascript"> 
    $(window).on('load', function () {
        if ((window.location.href.indexOf("livechat=true") != -1) {
            Moneypenny.startLink();
        }
    });
</script>

Proactive Chat

Place the script inside of the landing page where people will be landing from your email; for best results it is recommended that this goes as close as possible to the to the closing </body> tag. The url that they enter must contain "livechat=true&proactive" for example if your landing page is xyz.com/emailCampaign you will need to amend the link to xyz.com/emailCampaign?livechat=true&proactive

This script will open the chat as a proactive chat with the proactive message and time settings you have set globally in your portal.

<script type="text/javascript"> 
    $(window).on('load', function () {
        if (window.location.href.indexOf("livechat=true&proactive") != -1) {
            Moneypenny.openProactiveChat(true, true);
        }
    });
</script>

Proactive Chat (Custom message)

Place the script inside of the landing page where people will be landing from your email; for best results it is recommended that this goes as close as possible to the to the closing </body> tag. The url that they enter must contain "livechat=true&proactivecustom" for example if your landing page is xyz.com/emailCampaign you will need to amend the link to xyz.com/emailCampaign?livechat=true&proactivecustom

This chat will open your proactive chat with a custom message that can be set in the code here you can see it is set as "Hello, can I help you with our setup process?"

<script type="text/javascript">
    $(window).on('load', function () {
        if (window.location.href.indexOf("livechat=true&proactivecustom") != -1) {
            Moneypenny.openProactiveChat(true, true, 'Hello, can I help you with our setup process?');
        }
    });
</script>
